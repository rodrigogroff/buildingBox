Complete set of features, ready for customization:



Organization
Setup system environment variables and different program formats.
Create specific profiles, with CRUD grants per features and system specific views.
Create projects by selecting from a range of templates.
Assign a range of users into projects and set their roles inside the organization.
Create multiple phases inside your projects to organize your work throughout time.
During any phase of the project, create sprints/versions to produce releases.

Layout
Bootstrap 3, nextgen template.
Responsive for large and laptop monitors.
Gradient colorization, better visualization schemes.
Transparent web elements, accordion elements.

Security
Change and reset user passwords, easily.
Audit changes in important entities, verifying date, user and change details.
Control login time/IP for your users, avoiding security risks and threats.

News / Surveys
Dissemination of internal news for all users or for a specific project.
Use surveys to understand better the necessities of your team.

Templates
For simple or basic projects, we offer the 'Software Maintenance' template.
CMMI projects (levels 2 to 5), all key process areas (KPA) are monitored and checked.
For non-software project types, you can select 'Custom' and create/edit your own workflow.

Tasks
Create 'task types' to better organize and monitor your work inside projects.
Create 'task categories' within 'task types' to better sort project assessments.
Within a task category, create a unique workflow describing all aspects of work.
Within a task category, define accumulators to sum different types of units (hours, money).

Workflow
Workflows define how a task progresses from an open state.
A workflow has an order and the property to open/close the tasks.
Using appropriate labels, managers can see where in time a task is at.

Views
Check all tasks by using a 'Kanban' view, focusing on workflow aspects.
'Management' view, using workflow statistics (on active versions).
'Management' view, using project versions details (work and re-work %).
'Timesheet' view, showing dates and hours for users.
